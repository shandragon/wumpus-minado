# Wumpus Minado

Este é um jogo híbrido que combina elementos de Campo Minado e do Mundo de Wumpus. O objetivo é coletar todos os tesouros e encontrar a saída, evitando minas e um monstro.

## Requisitos

* Python 3
* PyGame

## Como Jogar

1. Crie e ative um ambiente virtual (recomendado):
   ```bash
   python3 -m venv venv
   source venv/bin/activate
   ```
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
   ```
3. Execute o jogo:
   ```bash
   python main.py
   ```
